Examples¶

cantera_demos.m: Run all examples

catcomb.m: Catalytic combustion of a stagnation flow on a platinum surface

conhp.m: ODE system for a constant-pressure, adiabatic reactor.

conuv.m: ODE system for a constant-volume, adiabatic reactor.

diffflame.m: - A non-premixed opposed-jet flame.

equil.m: a chemical equilibrium example.

flame.m: - create a flame object.

flame1.m: - A burner-stabilized flat flame

flame2.m: An axisymmetric stagnation-point non-premixed flame

ignite.m: Zero-dimensional kinetics: adiabatic, constant pressure.

ignite_hp.m: Solves the same ignition problem as ‘ignite’, but uses function

ignite_uv.m: Solves the same ignition problem as ‘ignite2’, except

isentropic.m: isentropic, adiabatic flow example

periodic_cstr.m: A CSTR with steady inputs but periodic interior state.

prandtl1.m: Prandtl number for an equilibrium H/O gas mixture.

prandtl2.m: Prandtl number for an equilibrium H/O gas mixture.

rankine.m: This example computes the efficiency of a simple vapor power cycle.

reactor1.m: Zero-dimensional kinetics: adiabatic, constant pressure.

reactor2.m: Zero-dimensional kinetics: adiabatic, constant volume.

reactor_ode.m: system for a generic zero-dimensional reactor.

run_examples.m: runs all examples

surfreactor.m: Zero-dimensional reactor with surface chemistry

test_examples.m: runs selected examples without pausing

transport1.m: mixture-averaged transport properties
