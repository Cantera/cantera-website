Detailed Description

The line shape for pure collisional broadening.

The Lorentzian line shape is

\[ L(\Delta\nu) = \frac{1}{\pi}\frac{\gamma}{\Delta\nu^2 + \gamma^2} \]

where \( \gamma = {\mbox{FWHM}/2} \).
