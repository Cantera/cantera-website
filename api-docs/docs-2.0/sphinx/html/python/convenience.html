Convenience Functions¶

Cantera.gases.Air()¶

Return a Solution instance implementing the O/N/Ar portion of reaction mechanism GRI-Mech 3.0. The initial composition is set to that of air

Cantera.gases.Argon()¶

Return a Solution instance representing pure argon.

Cantera.gases.GRI30(transport='')¶

Return a Solution instance implementing reaction mechanism GRI-Mech 3.0.
