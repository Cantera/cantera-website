Physical Constants¶

These values are the same as those in the Cantera C++ header file ct_defs.h.

cantera.avogadro¶

Avogadro’s Number, /kmol

cantera.gas_constant¶

The ideal gas constant in J/kmol-K

cantera.one_atm¶

One atmosphere in Pascals

cantera.boltzmann¶

Boltzmann constant

cantera.planck¶

Planck constant (J/s)

cantera.stefan_boltzmann¶

The Stefan-Boltzmann constant, W/m^2K^4

cantera.electron_charge¶

The charge on an electron (C)

cantera.electron_mass¶

The mass of an electron (kg)

cantera.faraday¶

Faraday constant, C/kmol

cantera.light_speed¶

Speed of Light (m/s).

cantera.permeability_0¶

Permeability of free space \(\mu_0\) in N/A^2.

cantera.epsilon_0¶

Permittivity of free space (Farads/m = C^2/N/m^2)
